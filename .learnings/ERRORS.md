# Error Log

Technical errors encountered and their fixes.

Format:
```
## [ERR-YYYYMMDD-NNN] service/component

**Logged**: ISO timestamp
**Severity**: low | medium | high | critical

### Error
Error message or description

### Cause
Root cause analysis

### Fix
What resolved it

### Prevention
How to avoid in future
```
