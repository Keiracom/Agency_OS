## Dev Review Process

| Step | Who | Action |
|------|-----|--------|
| 0 | CTO | Due diligence audit |
| 1 | Dev Team | Produce best work |
| 2 | CTO | Review (strengths + gaps) |
| 3 | Dev Team | Revise until approved |
| 4 | CTO | Code check |
| 5 | CTO | CEO report |

**Rule:** CEO says "continue" to advance steps.
