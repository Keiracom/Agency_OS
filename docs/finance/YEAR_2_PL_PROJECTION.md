## Part 2: Paddle.com Fee Structure

### 2.1 Paddle vs Stripe Comparison

| Fee Component | Stripe | Paddle |
|---------------|--------|--------|
| **Base Rate** | 2.9% | 5.0% |
| **Per Transaction** | $0.30 | $0.50 |
| **VAT/GST Handling** | You manage | Included |
| **Tax Filing** | You manage | Included |
| **Chargeback Protection** | Extra | Included |
| **Fraud Protection** | Extra | Included |
| **Multi-currency** | Extra fees | Included |

### 2.2 Paddle Fee Calculation

**Example: $2,500 AUD transaction**
- Paddle: ($2,500 × 5%) + $0.50 = **$125.50** (5.02%)
- Stripe: ($2,500 × 2.9%) + $0.30 = **$72.80** (2.91%)

**Difference:** $52.70 more per transaction

### 2.3 Why Paddle is Worth It for Y2

| Benefit | Value |
|---------|-------|
| No VAT registration in UK/EU | Saves $5,000-15,000/yr + accountant fees |
| No GST registration in AU/NZ | Already handled, but simplified |
| No sales tax nexus in USA | Saves $10,000-30,000/yr compliance |
| Single payout (weekly) | Simplified accounting |
| Handles refunds/chargebacks | Reduced support burden |
| **Net Benefit** | Saves $20,000-50,000/yr in compliance |
