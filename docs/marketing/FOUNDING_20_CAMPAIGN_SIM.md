## Part 5: Campaign Optimization Triggers

### Positive Signals (Scale)

| Signal | Action |
|--------|--------|
| Open rate > 50% on a subject line | Test variation, roll out to full list |
| Reply rate > 7% on a sequence | Add volume, reduce gaps between emails |
| LinkedIn post > 2K impressions | Boost similar content, engage all commenters |
| Demo show rate > 80% | Current process working, document and maintain |

### Warning Signals (Fix)

| Signal | Action |
|--------|--------|
| Open rate < 35% | Test new subject lines immediately |
| Reply rate < 3% | Review personalization quality, check spam scores |
| Demo no-show > 30% | Add SMS reminders, shorten gap between book/call |
| Unsubscribe > 2% | Review list quality, soften messaging |
| Zero replies from vertical | Drop vertical or test completely different angle |

### Emergency Signals (Stop & Review)

| Signal | Action |
|--------|--------|
| Spam complaints | STOP campaign, review all copy, check domain health |
| Angry replies > 5% | STOP, review targeting, likely wrong ICP |
| Domain blacklisted | STOP all sending, begin remediation |
| Zero demos Week 2 | Full strategy review, likely fundamental issue |
