# UI_UX_SPECIFICATION.md
**Status:** APPROVED
**Purpose:** Defines visual requirements for "The Glass Box" UI.

## 1. CO-PILOT COCKPIT
- Split-screen layout: Draft Email (Left) vs Source Data (Right).
- Source Data cards must show: Scraped LinkedIn Post, Website Value Prop, or ALS Score breakdown.

## 2. ALS SCORECARD
- Hover pop-over on score badges showing the 5-point breakdown (Data, Authority, Fit, Timing, Risk).

## 3. COMMAND CENTER
- Activity Ticker (Bloomberg style) on left.
- "Global Intelligence" widget showing platform-wide stats.
- "Monthly Capacity Gauge" (Fuel gauge) instead of credit burn.

## 4. VOICE TRANSCRIPTS
- Text-only chat bubbles for Voice AI logs. No audio player.
