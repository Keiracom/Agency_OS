# FCO-002 Decision — Smart Prompts + SDK Deprecation

**Date:** 2026-02-05
**Status:** ✅ RATIFIED
**Governance Event:** FCO-002

---

## Executive Summary

**Smart Prompting + Siege Waterfall = SDK Enrichment Deprecated**

The SDK Agent was designed before Siege Waterfall existed. Now redundant for data collection. Smart Prompting becomes primary for all content generation.

---

## The Decisions

### 1. SDK Architecture Change

| Component | Previous | Decision |
|-----------|----------|----------|
| **SDK Enrichment** (web search) | Active for Hot | ❌ **DEPRECATE** |
| **SDK Email** | Active for Hot | ❌ **REPLACE with Smart Prompts** |
| **SDK Voice KB** | Active for Hot | ❌ **REPLACE with Smart Prompts** |
| **Smart Prompts** | Active | ✅ **PRIMARY for all content** |
| **SDK Objection Handling** | Active | ✅ **KEEP** (10% Claude routing) |

**Rationale:** Siege Waterfall already collects funding, hiring, LinkedIn signals. Smart Prompts synthesizes this data at $0.02/lead vs SDK's $1.21/lead.
