### GMB Disambiguation Fix

### DFS Reviews API Pattern
`/live` endpoint returns 404. Must use async:
1. POST `business_data/google/reviews/task_post`
2. Wait 15s
3. GET `business_data/google/reviews/task_get/{task_id}`
Cost: $0.0015/domain

### Google Ads Transparency Scraper Fix
`get_breif_ads()` has KeyError bug on image ads. Use raw API response:
```python
data = {'f.req': '{"1":"' + adv_id + '","2":"' + cid + '","5":{"1":1}}'}
r = scraper.reqs.post('https://adstransparency.google.com/anji/_/rpc/LookupService/GetCreativeById', ...)
inner = r.json()["1"]
date_entries = inner.get("17", [])  # [{"1": location_code, "5": YYYYMMDD}]
```
