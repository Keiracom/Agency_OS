# --- Deployment Environment Gate ---
# [Rule] -> RULE 6 / GOV-12: Gates as code
# [Action] -> Add conditional check for ENVIRONMENT variable.
# [Rationale] -> Enforce production-only deployments as per directive, preventing accidental deployments to other environments.

**Explanation:**
1.  **`if [ -z "$ENVIRONMENT" ] || [ "$ENVIRONMENT" != "production" ]; then`**: This line checks two conditions:
    *   `[ -z "$ENVIRONMENT" ]`: Checks if the `ENVIRONMENT` variable is unset or empty.
    *   `[ "$ENVIRONMENT" != "production" ]`: Checks if the `ENVIRONMENT` variable is set but its value is *not* "production".
    *   The `||` (OR) operator means if *either* of these conditions is true, the `if` block will execute.
2.  **`echo "ERROR: Deployment blocked..."`**: Prints a clear error message to standard output.
3.  **`exit 1`**: Exits the script with a non-zero status code, indicating an error.
